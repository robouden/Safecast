Instructions

The software allows a driver to open the data file created by the bGeigie and see a map, some graphs and statistics.

It is a Java application, so Java needs to be installed first. Java can be downloaded here (but many computers have Java installed already).

The visualisation application jar file can be saved in any directory. No installation is required.

The application can be run under Windows by just double clicking on the .jar file.

When the application runs, it will pop up and ask the user to select a log file.

After a file is selected it will be displayed on the screen as a map, two graphs and a table. If the PC is not connected to the Internet, the map will not be displayed.

指示

このソフトはデータファイルを開けて、ｂガイギーで地図が見れたり、グラフや統計を見ることできます。

このソフトは、最初にJavaをインストールしてインストールする必要があります。Javaはここからダウンロードできます。（多くのコンピュータがすでにインストールされちます）

ファイルを選択した後で、スクリーンに地図と二つのグラフとテーブルが表示されます。もしそのときにＰＣがインターネットにつながっていなかったら、地図は表示されません。
